# DevOps Lab Portal Frontend - Next.js

This is the frontend UI for the DevOps Lab platform, powered entirely by React and Next.js 15.

## Prerequisites

- Node.js (`v18.17+`)
- The backend API must be running locally on port `8080`.

## Running the UI Locally

To start the Next.js development server:

```bash
npm install
npm run dev
```

The web UI will be accessible at:
`http://localhost:3000`
